# 🎬 TMDB Movie Fetcher - Complete Guide

## 🚀 Quick Start

1. Open `fetcher.html` in your browser (Chrome/Firefox recommended)
2. API key is pre-filled: `ba5dc12f58f09088d036049c565c2fe9`
3. Select mode:
   - **Test**: 5 movies (30 seconds)
   - **Small**: 20 movies (2 minutes)
   - **Full**: 3404 movies (3-4 hours)
4. Click **Start Fetching**
5. Files download automatically to your Downloads folder

## 📁 Downloaded Files

Files are saved as: `{movie-title-slug}-{tmdb-id}.json`

Example: `salaar-770906.json`

## 💡 Tips

- Keep browser window open during fetching
- Don't close the tab while running
- Test mode first to verify everything works
- Files save to your default Downloads folder

## 🐛 Troubleshooting

**Nothing downloading?**
- Check browser allows downloads
- Check Downloads folder
- Try Test mode first

**API errors?**
- Verify API key is correct
- Check internet connection
- Wait a few seconds and retry
